This function is used to check if the received rectangle fits on the screen.

The check consist of checking each corner of the rectangle with the resolution of the VGA screen. If the rectangle does not fit on the screen the function returns an error code.

This function writes the rectangle pixel data to the VGA RAM.

To write the pixel data to the VGA RAM the function is using the API_Draw_Line function. Using this function the four different lines of the rectangle can be drawn.

Attention
A weird bug which is solved with a hack. The right vertical line misses one pixel. To fix this the line is extended with one pixel.
