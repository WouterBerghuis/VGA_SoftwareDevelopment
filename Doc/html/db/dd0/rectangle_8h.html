Draw rectangle functions.

Draw rectangle functions.

Drawing the rectangle on the screen consist of the following steps:

  1. Check the received coordinates of the rectangle. Does the rectangle fit on the screen or is the size too big for the received position.
  2. Write the rectangle pixel data to the VGA RAM This is done with the use of the API_Draw_Line function
  3. Fill the inside of the rectangle if filled is equal to 1 This is done by setting the pixels within the rectangle corners to the received color.
Parameters
x_coorThis is the x-coordinate for the rectangle, (uint16_t)
y_coorThis is the y-coordinate for the rectangle, (uint16_t)
widthThis is the width of the rectangle, (uint16_t)
heightThis is the height of the rectangle, (uint16_t)
colorThis is the color with which the rectangle is drawn, (uint8_t)
filled0 = Not filled | 1 = Filled, if filled the inside of the rectangle will be filled with color, (uint8_t)

This function writes the rectangle pixel data to the VGA RAM.

To write the pixel data to the VGA RAM the function is using the API_Draw_Line function. Using this function the four different lines of the rectangle can be drawn.

Attention
A weird bug which is solved with a hack. The right vertical line misses one pixel. To fix this the line is extended with one pixel.
