◆ API_Draw_Bitmap()

BITMAP_ERROR_CODES API_Draw_Bitmap ( uint16_t  x_coor,
uint16_t  y_coor,
uint8_t  bitmap_nr 
)

This function is used for drawing a bitmap on the VGA screen.

The bitmap specified in bitmap_nr will be drawn from the position specified in x_coor and y_coor. These coordinates correspond with the topleft coordinate of bitmap.

The function will check if the bitmap fits on the screen before it starts drawing. If the bitmap doesnt fit on the screen the bitmap won't be drawn and the function returns with an error.

Parameters
x_coorThis the x-coordinate for the bitmap, (uint16_t)
y_coorThis the y-coordinate for the bitmap, (uint16_t)
bitmap_nrThis is the bitmap number, all bitmaps have an unique number
Return values
BITMAP_ERROR_CODES
