This function writes the rectangle pixel data to the V\+GA R\+AM. 

To write the pixel data to the V\+GA R\+AM the function is using the A\+P\+I\+\_\+\+Draw\+\_\+\+Line function. Using this function the four different lines of the rectangle can be drawn.

\begin{DoxyAttention}{Attention}
A weird bug which is solved with a hack. The right vertical line misses one pixel. To fix this the line is extended with one pixel.
\end{DoxyAttention}
