This function is used for drawing a bitmap on the V\+GA screen. 

The bitmap specified in {\ttfamily bitmap\+\_\+nr} will be drawn from the position specified in {\ttfamily x\+\_\+coor} and {\ttfamily y\+\_\+coor}. These coordinates correspond with the topleft coordinate of bitmap.

The function will check if the bitmap fits on the screen before it starts drawing. If the bitmap doesnt fit on the screen the bitmap won\textquotesingle{}t be drawn and the function returns with an error.
